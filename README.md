## 🇹🇷 Türkçe

### 🤔 Kimler Neden Kullanmalı?
Cordiss; Yayıncılar, Yazılımcılar, Oyuncular ve VPN/DNS/WARP gibi harici sistemlerle uğraşmadan Discord'a kesintisiz erişmek isteyen herkes için tasarlanmıştır.

Diğer bypass yöntemlerinin aksine **tamamen izole çalışır**. Sadece kendi yerel proxy'si üzerinden geçen Discord trafiğini yönetir ve bilgisayarınızdaki diğer hiçbir internet trafiğine müdahale etmez. 

**Bu sayede:**
* **Oyuncular için:** Ping, lag, "Robot musun?" doğrulamaları, "Bağlanılamıyor" hataları veya VPN tespit engelleri yaşanmaz.
* **Yayıncılar için:** Bilgisayarın genel internet trafiği etkilenmediği için yayınlarda gecikme (delay), yavaşlık veya drop sorunları oluşmaz.

### ✨ Özellikler
* Discord Web versiyonunda yer alan tüm güncel özellikler.

To build or develop the project locally, follow these steps:

    Clone or download this repository:
    Bash

    git clone [https://github.com/RenYukki/Cordiss.git](https://github.com/RenYukki/Cordiss.git)
    cd Cordiss

    Install the required dependencies:
    Bash

    npm install

    Start the application in development mode:
    Bash

    npm start

    To generate the installer or production binary:
    Bash

    npm run dist
