## Kimler neden kullanmalı?
-> Cordiss Yayıncılar, Yazılımcılar, Oyuncular; Vpn,dns,warp gibi sistemlerle uğraşmadan discorda girmek isteyen, gizliliğine ve güvenliğine önem veren herkes kullanabilir.
Cordiss Diğer discorda bağlanma yöntemlerinin aksine tamamen izole çalışır, kendi yerel proxysindeki discord trafiği hariç hiçbir internet trafiğine karışmaz bu sayede:
Oyun oynarken Ping/lag/Robot musun?/Bağlanılamıyor/vpn tespit edildi gibi problemlerle karşılaşmıyorsunuz. Yayın yaparken gecikme yavaşlık drop gibi problemlerle karşılaşmıyorsunuz çünkü cordiss bilgisayarınızdaki diğer trafiklere karışmaz.

## Nerdler için / For nerds:


**Nasıl Build edilir?**
1. Bu depoyu indirin veya kopyalayın.
2. Klasörde terminal açıp `npm install` yazın.
3. Uygulamayı başlatmak için `npm start` yazın.
4. Setup oluşturmak için `npm run dist` yazın. 
 
**How to Build?**
1. Download or copy this repo.
2. Open terminal in your folder and type `npm install`.
3. To start the app type  `npm start`.
4. To create setup or binary `npm run dist`
